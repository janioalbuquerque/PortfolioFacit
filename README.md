# Portifolio-Facit

Projeto de um portifolio, inicialmente para o curso de ADS da Faculdade de Ciência do Tocantins - FACIT, onde os responsaveis pelo portfólio do curso, vão poder postar os trabalhos dos acadêmicos para ficarem guardados e servir de portifolio para o curso de ADS da FACIT, para que pessoas ou interessados no curso possam ver os trabalhos feitos pelos academicos.
